# Dota 2 翻译小助手 v2.0.0

Dota2游戏翻译工具，支持实时翻译和手动翻译两种模式。

## 功能特点

### 模式说明

| 功能 | 说明 |
|------|------|
| **实时翻译** | 自动识别屏幕上指定区域的英文，翻译成中文以弹幕形式显示 |
| **手动翻译** | F6快捷键触发，翻译聊天框内容 |

### 实时翻译功能

- ✅ **实时识别** - 持续监控屏幕指定区域
- ✅ **弹幕显示** - 翻译结果以透明弹幕形式显示在屏幕右侧
- ✅ **自动清理** - 显示2条弹幕，新的来时最老的消失
- ✅ **Dota2词库** - 智能翻译游戏术语：
  - 英雄简称：QOP→女王、PA→幻刺、AM→敌法、SF→影魔等
  - 物品名称：bkb→黑皇杖、dd→双刀、roshan→肉山等
  - 游戏术语：gank→抓人、push→推塔、mid→中路等
- ✅ **智能去重** - 避免重复翻译相同内容
- ✅ **位置记忆** - 识别区域和弹幕位置自动保存

### 手动翻译功能

| 模式 | 快捷键 | 风险 | 说明 |
|------|--------|------|------|
| **非严格** | F6 | ⚠️ 低风险 | 自动翻译聊天框内容并发送，可能影响游戏操作 |
| **严格** | F6 | ✅ 零风险 | 呼出悬浮窗，手动复制翻译结果，不抢焦点 |

> **重要提示**：
> - **非严格模式**：自动操作剪贴板和模拟键盘，有较低风险影响游戏（如打字时误触发）
> - **严格模式**：悬浮窗不抢焦点，Enter键翻译后复制到剪贴板，完全不干扰游戏

### 其他功能

- ✅ **现代化GUI界面** - 简洁清晰
- ✅ **系统托盘支持** - 可最小化到托盘
- ✅ **严格模式悬浮窗** - 透明置顶，不抢焦点

## 快速开始

### 方式1：直接使用EXE（推荐）⭐

**适合所有用户，无需Python环境**

1. **下载EXE文件**
   - 从 [Releases](../../releases) 下载 `Dota2Translator.exe`

2. **运行程序**
   ```
   右键 Dota2Translator.exe → 以管理员身份运行
   ```
   > ⚠️ **重要**：必须以管理员身份运行，否则在游戏中快捷键可能无效

3. **使用实时翻译**
   - 点击"设置区域"按钮，框选需要翻译的屏幕区域
   - 点击"开始实时翻译"按钮启动
   - 翻译结果将以弹幕形式显示

4. **使用手动翻译**
   - 点击"非严格"或"严格"按钮切换模式
   - 点击快捷键按钮可自定义F6
   - 点击"禁用"可暂停翻译功能

### 方式2：使用源码运行

**适合开发者**

```bash
pip install -r requirements.txt
# 以管理员身份运行 PowerShell/CMD 后执行：
python src/dota2_translator_gui.py
```

## 使用方法

### 实时翻译
1. 点击"设置区域"按钮
2. 在屏幕上框选需要翻译的区域（如游戏聊天区域）
3. 点击"开始实时翻译"按钮
4. 区域内的英文将被实时识别并翻译成中文
5. 翻译结果以弹幕形式显示在屏幕右侧

### 非严格模式
1. 确保在"非严格"模式下（按钮显示"严格"表示未启用）
2. 进入Dota2游戏，打开聊天框
3. 输入中文：`帮我买眼`
4. 按 **F6** 键
5. 自动翻译并发送：`Help me buy wards`

### 严格模式
1. 点击"严格"按钮切换到严格模式
2. 按 **F6** 呼出悬浮窗（不会打断游戏）
3. 输入中文，按 **Enter** 翻译
4. 结果自动复制到剪贴板
5. 切换回游戏粘贴使用

## 常见问题

### Q: 实时翻译不工作？
- 确保以管理员身份运行
- 点击"设置区域"正确框选翻译区域
- 确保框选区域内有英文文本

### Q: 按触发键没反应？
- **确保以管理员身份运行**（最重要！）
- 确保程序已启动且状态显示"已启动"
- 确保Dota2窗口在前台
- 确保聊天框已打开（非严格模式）

### Q: 严格模式悬浮窗不出现？
- **确保以管理员身份运行**
- 点击"严格"按钮启用严格模式
- 按F6呼出悬浮窗

### Q: 悬浮窗位置不保存？
- 关闭时位置会自动保存
- 重启程序后会恢复上次位置

### Q: 游戏中快捷键无效？
- 必须以管理员身份运行程序
- 右键EXE → 以管理员身份运行

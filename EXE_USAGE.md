# Dota 2 翻译器 - EXE版使用说明

## 📦 快速开始（无需Python）

### 方法1：直接运行EXE（推荐）

1. **下载文件**
   - 从GitHub Releases下载 `Dota2_Translator.exe`
   - 或从 `dist` 目录获取

2. **双击运行**
   - 双击 `Dota2_Translator.exe`
   - 看到命令行窗口显示"等待进入Dota2..."即成功

3. **进入Dota2游戏**
   - 打开聊天框（按Enter）
   - 输入中文：`帮我买眼`
   - 按 **F6** 键
   - 自动翻译并发送：`Help me buy glasses`

### 方法2：自定义配置后运行

1. **首次运行会自动生成配置文件**
   - 运行程序后会生成 `config.json`

2. **编辑配置文件（可选）**
   ```json
   {
       "trigger_key": "f6",
       "toggle_hotkey": "ctrl+alt+t"
   }
   ```

3. **修改触发键**
   - 将 `"trigger_key": "f6"` 改为你想要的键
   - 例如：`"f7"`, `"space"`, `"enter"`

4. **重新启动程序**

---

## ⌨️ 快捷键说明

| 按键 | 功能 | 说明 |
|------|------|------|
| **F6** | 触发翻译 | 在Dota2聊天框输入中文后按此键 |
| **Ctrl+Alt+T** | 开启/关闭 | 切换翻译器开关 |

## 🔧 常见问题

### Q1: 按F6没有反应？
- 确保Dota2窗口在前台
- 确保聊天框已打开
- 确认输入了中文内容
- 尝试按 Ctrl+Alt+T 检查是否开启

### Q2: 想更换触发键？
编辑 `config.json` 文件：
```json
{
    "trigger_key": "f7"
}
```

支持的按键：`f1`~`f12`, `a`~`z`, `0`~`9`, `space`, `enter` 等

### Q3: 杀毒软件报毒？
- 这是误报，因为程序使用了键盘模拟功能
- 添加到信任列表即可

### Q4: 需要联网吗？
- 需要！翻译功能调用在线API
- 首次翻译可能需要1-2秒
